# Planteer

Planteer is a responsive Django website for discovering and exploring real plants.

The website allows users to:

- Browse all plants
- View plant details
- Search and filter plants
- Add, update, and delete plants
- View related plants
- Send messages through the contact page
- Add comments to plants


## Technologies Used

- Python
- Django
- HTML
- CSS
- JavaScript
- Bootstrap

## Features

- Responsive design
- Arabic and English support
- Plant filtering by category and edibility
- Related plants section
- Frontend and backend validation
- Template inheritance
- Dark mode support
